# Smart-Tool-Agent

## Overview
Smart-Tool-Agent is a Python application that helps users with toll management and intelligent tool-based operations.

## Features
- Intelligent tool agent
- Dashboard interface
- Backend API
- ChromaDB integration

## Technologies Used
- Python
- Streamlit
- ChromaDB
- Google Gemini API

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
